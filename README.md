# DAO Toolkit

_DAO Toolkit_ is an easy-to-use desktop application for managing your _Dragon Age: Origins_ mods.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later)
- [Rust](https://www.rust-lang.org/)
- [pnpm](https://pnpm.io/)

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/dao-toolkit.git
    ```
2.  Install the dependencies:
    ```bash
    pnpm install
    ```
3.  Run the development server:
    ```bash
    pnpm tauri dev
    ```

## Tech Stack

- **Frontend:** [React](https://reactjs.org/), [TypeScript](https://www.typescriptlang.org/), [Vite](https://vitejs.dev/), [Tailwind CSS](https://tailwindcss.com/)
- **Backend:** [Rust](https://www.rust-lang.org/), [Tauri](https://tauri.app/)
